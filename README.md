# ok2-frontend

The frontend of Open Knesset v2.0

## Project setup

```shell
npm install
```

### Compile and hot-reloads for development

```shell
npm run serve
```

### Compile and minify for production

```shell
npm run build
```

### Lint files

Just lint:

```shell
npm run lint
```

Lint and fix errors:

```shell
npm run lint --fix
```
